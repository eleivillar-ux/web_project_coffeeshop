# Triple Espresso

Este es un proyecto de cafetería en linea, en donde ademas de poder generar reservaciones, puedes visualizar contenido multimedia sobre formas de preparar tu propio café en casa.

Las tecnologías utilizadas en este proyecto son HTML y CSS de la mano de la metodología BEM para un uso organizado y efectivo de las fuentes de estilo, haciendo uso de elementos como iframe para embeber contendio de otras paginas y hacer mas atractiva la interacción de los usuarios, de igual forma se hicieron uso de links que permiten que la navegación dentro de los apartados del sitio faciliten el proceso de reservación o contacto que se requieran.

Planes de mejora: Cambiar los estilos de la pagina basados en la colorimetría del mercado objetivo con el fin de hacer la pagina más atractiva para los usuarios, mejorar la distribución del contenido en la pagina y conectar la pagina con otros sitios en internet relacionados con el tema principal de interés y poder generar ordenes en línea de los productos de la cafetería.
